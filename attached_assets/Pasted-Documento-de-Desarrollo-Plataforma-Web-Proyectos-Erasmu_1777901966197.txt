Documento de Desarrollo: Plataforma Web Proyectos Erasmus+ IES Manuel Martín GonzálezEste documento contiene las instrucciones precisas para que Replit Agent (o un desarrollador) construya una plataforma web profesional, dinámica y autogestionable.1. Concepto del ProyectoCrear una aplicación web "Full-Stack" con una parte pública de alto impacto visual y un panel de administración privado para gestionar contenidos, proyectos, centros asociados y el calendario de movilidades.2. Instrucciones de Diseño (UI/UX)Estilo: Moderno, limpio y profesional ("Estilo Apple/SaaS").Paleta de Colores: - Azul Institucional (#003399 - Azul UE)Verde Sostenibilidad (#2D5A27)Blanco y Gris muy claro para fondos.Componentes Visuales:Hero: Pantalla completa con títulos animados (Framer Motion).Tarjetas: Bordes redondeados (rounded-2xl) y sombras suaves.Imágenes: Uso de placeholders de Unsplash dinámicos (ej: https://source.unsplash.com/featured/?education,nature).Interactividad: Los mapas deben ser fluidos y las transiciones entre páginas suaves.3. Arquitectura Técnica (Tech Stack)Frontend: React.js con Tailwind CSS.Backend: Node.js (Express).Base de Datos: SQLite (ideal para Replit) con Prisma ORM.Mapas: Leaflet.js (Librería gratuita de mapas interactivos).Iconos: Lucide-React.4. Estructura de Datos (Base de Datos)La base de datos debe contemplar las siguientes tablas:Settings: Título del sitio, redes sociales del IES, email, teléfono.Partners (Centros): Nombre, OID, País, Ciudad, Coordenadas (Lat/Lng), URL Web, Redes sociales del socio, Logo.Mobilities: Centro destino, fecha inicio, fecha fin, tema (WP), imagen de cabecera.Activities: Título, descripción, movilidad asociada.Media: Galería de imágenes y vídeos.5. Datos Reales para Configuración Inicial (Seed Data)Proyecto Principal: "The small big changes" (SEA)Coordinador: IES Manuel Martín González (Guía de Isora, España).Socios y Movilidades:PaísCentroTema (Work Package)Fecha EstimadaEspañaIES Manuel Martín GonzálezWP2: Uso del Plástico10/11/2025 - 14/11/2025TurquíaKaratay Mehmet-Hanife YapiciWP3: Tecnología y Móviles09/03/2026 - 13/03/2026LetoniaValmiera State GymnasiumWP4: Agua y Madera18/05/2026 - 22/05/2026RumaníaLiceul Teoretic "Traian Vuia"WP5: Moda Rápida (Fast Fashion)16/11/2026 - 20/11/2026PortugalES de Emídio NavarroWP6: Energía22/02/2027 - 26/02/2027Macedonia"Sv.Kliment Ohridski" High SchoolWP7: Dieta y Salud24/05/2027 - 28/05/20276. Funcionalidades del Panel de Control (Admin)El panel debe ser accesible mediante /admin con un sistema de login seguro.Dashboard: Resumen de próximas movilidades.Gestor de Socios: Formulario para añadir/editar los centros de Macedonia, Portugal, etc., incluyendo sus coordenadas para que aparezcan en el mapa.Editor de Contenidos: Poder cambiar los textos de la página de inicio sin saber programar.Gestor de Movilidades: Calendario interactivo donde se arrastran o editan las fechas de viaje.Subida Multimedia: Capacidad de reemplazar las fotos de stock por fotos reales de los viajes una vez realizados.7. Instrucciones para Replit Agent (Prompt)"Construye una aplicación Full-Stack con React y Node.js. El objetivo es una web para el IES Manuel Martín González que promocione sus proyectos Erasmus+.Características obligatorias:Mapa Interactivo: Usa Leaflet para mostrar los 6 países del proyecto SEA. Al clicar en un país, debe abrirse un popup con la info del centro destino y su movilidad.Landing Page: Sección Hero impactante, Timeline de movilidades, Galería de fotos y sección de Actividades detalladas por cada centro.Admin Panel: Crea una interfaz CRUD completa para gestionar los socios, las fechas de los viajes y las actividades. Usa una base de datos SQLite para persistir los datos.Datos iniciales: Carga los datos del proyecto 'The small big changes' proporcionados en la tabla de socios.Estética: Usa Tailwind CSS con una estética premium, tipografías sans-serif modernas y animaciones de entrada.Responsivo: La web debe verse perfecta en móviles para que los alumnos y familias la consulten desde sus teléfonos."
